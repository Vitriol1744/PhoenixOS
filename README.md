Modern Unix-like Operating System written in C++
## Supported architectures
- x86-64

## Requirements:
- nasm
- clang
- ld, lld
- make
- xorriso 
- qemu
#
## How to build?
Simply by doing:
```
make
```
### How to run?
```
make qemu
```
